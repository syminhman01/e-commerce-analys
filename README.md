# E-Commerce SQL Analysis

## Project Overview
This project analyzes an e-commerce sales dataset using SQL to extract business insights.

## Dataset
The dataset contains:
- customers
- products
- regions
- sales transactions

## Key Analyses
1. Overall business metrics
2. Sales by category and region
3. Top products and customers
4. Product contribution within category
5. Month-over-month revenue growth
6. Most profitable category by region

## SQL Skills Used
- JOIN
- GROUP BY
- CTE
- Window Functions
- LAG
- RANK / ROW_NUMBER
- Time-based analysis

## Key Insights
- Body care dominates profitability across most regions
- Revenue growth shows seasonal patterns
- A small set of products contributes significantly to total revenue
